# 리눅스
## 사용자 권한
리눅스에서는 파일과 디렉토리에 접근을 제어  
`ls -l`로 권한 확인 가능  
  
> ex) `drwxrw-r--`
> d : 디렉토리인지 파일인지 구별  
> r : 읽기 권한(4) w : 쓰기 권한(2) x : 실행 권한(1)  
> 3개씩 나눠서 소유자(u), 그룹(g), 그외의 사용자들(o)에 대한 권한  
  
`chmode`로 권한 추가, 수정, 삭제 가능  
*ex) chmode u+rw [파일명] //심볼릭*  
    *chmode 753 [파일명] //8진수*  
  
+) `chown`로 파일과 디렉토리의 소유자, 그룹 변경
