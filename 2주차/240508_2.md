# 데이터베이스
## DB 이중화
DB 서버에 장애가 생겨 서비스에 데이터가 들어오지 않게 됨으로 DB를 복제해 서비스 장애 시간을 줄이는 방법  

## Master-Slave 구성
2대 이상의 DBMS에 데이터를 저장하는 비동기 방식의 복제 방식  
ex) Replication 방식  
  
### MySQL Replication 
1. master에서 db를 백업하고 slave로 송신해 db를 복제  
2. 이후에는 수정사항을 기록해 log 전송해 db 동기화  
  
 ⇒ master는 읽기, 쓰기 역할 slave는 읽기 역할
